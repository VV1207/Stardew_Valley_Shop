# 🌾 星尔玛商店 — 星露谷物语主题网购平台

一个以《星露谷物语》(Stardew Valley) 为主题的全功能在线购物商店网站，包含完整的商品浏览、购物车、订单管理、村民配送、用户系统、新闻资讯、社区反馈等功能模块。

## ✨ 功能特性

### 🛒 商店首页 (index.html)
- **800+ 种星露谷物品**：涵盖作物、动物、果树、工匠制品、矿物、渔具、武器、帽子等 20+ 分类
- **搜索与筛选**：按名称实时搜索、按分类快速筛选
- **品质选择**：普通/银星/金星/铱星多品质切换，价格自动更新
- **智能购物车**：右侧边栏实时显示、数量调节、一键移除、小计计算
- **结账明细**：订单表格预览、村民喜好统计、一键创建订单
- **收藏功能**：收藏/取消收藏喜爱的物品（后端持久化）
- **村民赠送反应**：查看每个物品的村民喜好（最爱 ❤️ / 喜欢 👍 / 一般 ➖ / 不喜欢 👎 / 讨厌 💔）
- **智能推荐栏**：精品推荐（价格前10随机5）、村民最爱、限时热卖
- **分页导航**：每页 16 件商品，支持页码跳转
- **金币图标**：全局统一使用 Gold.png 金币图标

### 📰 星闻露谷 (news.html)
- **三大新闻栏目**：鹈鹕日报 🐦、离地而居 🏡、酱料女皇 👑
- **鹈鹕日报**：毕业特刊、春季播种、钓鱼大赛、秘密蘑菇等新闻卡片
- **离地而居**：四季农业小提示（16期），图文并排展示
- **酱料女皇**：32 道烹饪食谱卡片，含季节标签、原料清单、食谱描述
- **新闻详情模态框**：点击标题展开完整报道
- **互动系统**：点赞 👍、留言 💬（后端持久化，支持所有用户）
- **侧栏导航**：左栏切换栏目，右栏显示内容

### 📦 配送服务 (delivery.html)
- **到店自提 🏪**：5 分钟倒计时，显示预计可取时间
- **村民配送 🚚**：38 位可选村民配送员，每位有独特趣味寄语
- **配送地址**：默认鹈鹕镇事务大厅，可自由修改
- **结果弹窗**：配送确认信息展示
- **预期送达时间**：根据配送模式自动计算

### 👤 个人中心 (profile.html)
- **我的收藏**：网格展示收藏物品，含图片、名称、分类、价格
- **历史订单**：订单列表含搜索、状态标签、评价系统
- **评价系统**：收货服务 ⭐ / 商品品质 🏷️ / 综合评分 ⭐ 三星评价，支持修改
- **用户反馈**：社区反馈墙（展示所有用户留言）+ 反馈提交表单

### 🔑 用户系统 (login.php / register.php)
- **注册/登录**：含图形验证码，防止机器人
- **会话管理**：PHP Session 持久化，跨页面状态同步
- **用户信息栏**：导航栏右侧显示用户名、下拉菜单（个人中心/退出登录）

### 🎵 背景音乐
- **跨页面连续播放**：使用 localStorage 同步播放状态、当前曲目、播放位置
- **3 首原声音乐**：Cloud Country、Stardew Valley Overture、Summer (Nature's Crescendo)
- **唱片旋转动画**：播放时唱片图标旋转，暂停时停止
- **手动触发播放**：不自动播放，用户点击唱片按钮开始播放

## 🚀 快速开始

### 环境要求

- [XAMPP](https://www.apachefriends.org/) (Apache + MySQL + PHP)
- PHP 7.4+
- MySQL 5.7+

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/VV1207/Stardew_Valley_Shop.git
   ```
   将项目放到 XAMPP 的 `htdocs` 目录下。

2. **启动服务**
   - 启动 XAMPP 的 Apache 和 MySQL 服务

3. **配置数据库**
   - 打开 phpMyAdmin: http://localhost/phpmyadmin
   - 导入 `setup_db.sql` 创建数据库和所有表
   - 如需修改数据库连接信息，编辑 `db_config.php`

4. **访问项目**
   ```
   http://localhost/Stardew_Valley_Shop/
   ```

## 🗄️ 数据库结构

| 表名 | 用途 |
|------|------|
| `users` | 用户账号（用户名、邮箱、密码哈希、注册时间） |
| `cart` | 购物车（用户关联的物品数量与品质） |
| `favorites` | 收藏夹（用户收藏的物品 ID） |
| `orders` | 订单（订单号、物品明细、总价、配送方式） |
| `order_items` | 订单明细（物品、数量、单价、品质） |
| `reviews` | 订单评价（配送服务评分、商品品质评分、综合评分、评论） |
| `user_feedback` | 用户反馈（反馈类型、内容） |
| `news_interactions` | 新闻点赞（新闻 ID、用户 ID） |
| `news_comments` | 新闻评论（新闻 ID、用户 ID、评论内容） |
