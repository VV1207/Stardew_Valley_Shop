🧩 组件清单 Components

📌 统计状态栏

展示总物品数、最爱总数、讨厌总数三个徽章。数据从 items_data.js 自动计算。

静态auto-compute

🔍 搜索框

输入关键词实时过滤物品列表。匹配物品的 name 字段（中文名）。

实时过滤input事件

🏷️ 分类标签页

19 个分类按钮 + "全部"。点击切换 activeCategory，重新渲染商品网格。

19分类切换过滤

🃏 商品卡片

展示图片、中文名、分类标签、来源、价格、星级选择器、数量加减及小计。点击进入详情弹窗。

响应式网格星级联动

⭐ 星级选择器

仅对有 has_quality 的物品显示。选项：普通/银星/金星/铱星，切换时价格联动更新。

品质价格联动

🔢 数量控制器

− / 数字输入 / + 三部件。范围 0~999。变更时更新小计和购物车状态。

加减按钮手动输入

🛒 购物车栏

显示当前总件数、总金额、结账按钮、清空按钮。任何数量变动后自动更新。

实时总计浮动挂载

💬 物品详情弹窗

点击卡片打开。展示完整信息（品质选择、添加数量 + 加入购物车）、村民赠送喜好表。

模态框5级好感

🧾 结账弹窗

展示订单明细、总计金额。底部配送方式选择 —— "🏪 自提" / "🚚 配送"。选择配送后展开村民选择面板。

配送选择村民联动

📢 Toast 提示

底部弹出短暂提示。配送相关提示："已选择 XX 配送"、"配送员已出发！"等。

通知2s消失

🔐 登录弹窗

首次访问页面时自动弹出（若 localStorage 无用户信息）。表单字段：用户名（必填）、住址（选填，默认"鹈鹕镇事务中心"）、联系电话（必填）。提交后存入 localStorage。

localStorage表单验证

👤 用户信息栏

登录后在页面右上角固定显示。展示：头像图标 + 用户名 + 住址缩写。鼠标悬停可查看完整信息。点击可退出登录。

固定定位hover卡片

🚚 配送方式选择

结账弹窗底部配送方式切换。两个选项：🏪 自提（免费，到店取货）/ 🚚 配送（选择村民 + 配送到住址）。

单选切换联动村民

👥 村民配送员选择

选择"配送"后展开村民网格面板。支持：所有村民头像 + 姓名展示、点击选中高亮、显示配送地址（用户住址）、村民趣味配送寄语。

网格选择配送寄语

🔄 用户交互流程 UX Flow

🔐 首次打开页面 → 检测 localStorage → 未登录 → 弹出登录弹窗 → 填写 用户名 / 住址 / 电话 → 提交 → 存入 localStorage → 右上角显示用户信息
🌱 加载 items_data.js → 渲染分类按钮 + 商品网格
🔍 搜索 / 点分类 → 实时过滤商品列表 → 重新渲染网格
🖱️ 调整数量 +/- → 更新购物车数组 → 同步小计 + 购物车栏
⭐ 切换品质 → 更新单价显示 → 联动小计 + 购物车价格
📦 点击商品卡片 → 打开详情弹窗 → 查看村民喜好 + 加入购物车
✨ 点击结账 → 打开订单明细弹窗 → 选择 自提 / 配送 → 若配送 → 选择村民 → 确认 → 显示配送信息 → 清空购物车 → 提示

💾 数据架构 Data Flow

ITEMS_DATA (全局)  // 374 items
│
├── item { id, name, category, price, has_quality, qualities, image, ... }
│
├── USER_INFO (localStorage)
│   { username, address, phone }
│   address 默认 "鹈鹕镇事务中心"
│
├── cartItems[] (运行时状态)
│   { itemId, name, quality, quantity, price, ... }
│
├── delivery (运行时状态)
│   { mode: 'pickup' | 'delivery', villager: '阿比盖尔' }
│
└── VILLAGERS (常量)
    ['亚力克斯','艾利欧特','哈维','山姆','塞巴斯蒂安','谢恩',
    '阿比盖尔','艾米丽','海莉','莉亚','玛鲁','潘妮', ... 共30+位]

状态管理：纯前端 JS + localStorage。用户信息持久化存储，购物车和配送选项为运行时内存状态。
